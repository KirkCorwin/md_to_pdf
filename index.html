Markdown → PDF

Paste Markdown, pick a layout, download a US Letter PDF. Output is a real text PDF (pdfmake), not a screenshot—better for ATS parsers than HTML-to-canvas tools. Resume (compact) uses narrow margins and moderate spacing; Resume (ultra compact) tightens further for one-page fits.

Enter generates and downloads the PDF (same as Download PDF). Shift+Enter inserts a line break. While the PDF is building, the editor is briefly read-only.

Uses marked + pdfmake from jsDelivr. For GitHub Pages, keep these script tags or vendor the files into this folder. If file:// blocks scripts, run python -m http.server 8765 here and open http://localhost:8765.

Formatting guide

Education section (Resume compact / ultra compact only)

Joining and left/right layout run only under ## Education (also ## Education and Training, ## Academic Background, ## Academics). Other sections are not rewritten.

Typical pattern (two or three lines per school):

  1. Institution line — school, location, degree (e.g. State University — City, ST — B.S. Major). Use — or | separators.
  2. Meta line — dates and GPA (e.g. Expected graduation: May 2027 | GPA: 3.82 or Graduated: June 2023).
  3. Detail line (optional) — Focus:, Relevant coursework:, honors, etc.

Compact puts institution (left) and meta (right) on one PDF row; detail stays on the next line. Ultra compact also pulls detail onto the right column to save height.

Auto-join runs only when the first line looks like an institution and the next like meta. Otherwise lines stay as written.

Optional HTML comments (stripped from the PDF):

  • <!-- edu-join-next --> — force the next line onto the right column
  • <!-- edu-join-next:2 --> — join the next two lines to the right
  • <!-- edu-no-join --> — skip auto-join for this line and the next
  • <!-- edu-compact-off --> — disable education joins for that section
General Markdown notes
  • Section titles: ## Section name (only ## headings are styled as section headers).
  • Bullets: lines starting with - (space after the hyphen).
  • Links: [label](https://example.com) — rendered as clickable text in the PDF.
  • Hard line break inside a paragraph: end a line with two spaces, then newline (standard Markdown).
  • Blank line between blocks starts a new paragraph.
  • Cover letter and Resume modes use normal spacing and do not run education preprocessing.
  • Prefer plain text in the body (avoid **bold** / *italic*) for neutral, ATS-friendly typography.
